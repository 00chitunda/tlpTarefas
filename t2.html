CONTEÚDO: INTRODUÇÃO AO CSS

O que é CSS?

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata).


O CSS é uma linguagem usada para dar estilo às páginas Web criadas com HTML.
Enquanto o HTML serve para estruturar o conteúdo (textos, imagens, botões), o CSS é responsável pela aparência visual da página.

Exemplos do que o CSS faz:

    Mudar cores dos textos;
    Alterar o tamanho das letras;
        Definir cores de fundo;
        Posicionar elementos na página.
        Posicionar elementos na página.

      Sem CSS, as páginas Web ficam simples, sem cores e pouco atractivas.


      Importância do CSS no Desenvolvimento Web

      O CSS é muito importante porque:

      • Melhora a aparência dos sites, tornando-os mais bonitos e organizados;

      • Facilita a leitura do conteúdo;

      • Permite a separação entre conteúdo e design:

      o HTML → conteúdo

      o CSS → aparência

      • Ajuda a manter e actualizar os sites com mais facilidade;

      • Garante uma melhor experiência do utilizador.

      Hoje em dia, praticamente todos os sites usam CSS.


      Tipos de Folhas de Estilo CSS

      Existem três formas principais de aplicar CSS em uma página HTML:

      CSS Inline

      O CSS inline é aplicado directamente dentro da tag HTML. Exemplo:

      Este texto é azul

      Características: • É simples e rápido; • Serve para pequenas alterações; • Não é recomendado para páginas grandes.

      CSS Interna (Internal CSS)

      O CSS interno é escrito dentro da própria página HTML, usando a tag

      Características:

      • Organiza melhor o código que o CSS inline; • Aplica estilo a vários elementos ao mesmo tempo; • É usado quando o estilo é apenas para uma página 3.3 CSS Externa (External CSS) O CSS externo é escrito num ficheiro separado com a extensão .css.

      Exemplo do ficheiro estilo.css:


      p { color: red; } Ligação no HTML:

      Características:

      • É a forma mais usada e recomendada; • Permite reutilizar o mesmo estilo em várias páginas; • Facilita a manutenção do site. 4.
